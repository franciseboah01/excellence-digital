EXCELLENCE DIGITAL CENTER basé à Korhogo / Sirasso.

## 🎯 Description

EXCELLENCE DIGITAL CENTER est un centre spécialisé dans :
- 💼 Les services bureautiques
- 🌐 Les solutions digitales & design
- 💻 Le développement web & mobile
- 🎓 La formation pratique

---

## 👥 Rôles utilisateurs

| Rôle | Accès |
|---|---|
| Visiteur | Consultation + Demande de service |
| Client | Services + Formations + Espace perso |
| Enseignant | Ressources + Notifications apprenants |
| Admin | Supervision totale |

---

## 🛠️ Stack Technique

- **Backend** : PHP 8.x / Laravel 12
- **Frontend** : Blade + Tailwind CSS
- **Base de données** : MySQL 8
- **Auth** : Laravel Breeze
- **Rôles** : Spatie Laravel Permission
- **Emails** : Laravel Mail
- **Fichiers** : Laravel Storage

---

## ⚙️ Installation

```bash
git clone https://github.com/TON_USERNAME/excellence-digital.git
cd excellence-digital
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
npm run dev
php artisan serve
```
